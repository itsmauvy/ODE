Jost

RESPONSIVE WEB DESIGN

모든 화면에서
같은 브랜드 경험을

PC와 모바일 환경 모두를 고려한
반응형 웹사이트로 설계했습니다.

어떤 디바이스에서도 ODE의 감성과
브랜드 무드가 일관되게 전달될 수 있도록
화면 구성을 최적화했습니다.

PC Mobile Responsive Layout Publishing

01

Main Page

사용자가 브랜드의 감성과 분위기를 먼저 경험한 뒤,
자연스럽게 제품 탐색으로 이어질 수 있도록
콘텐츠 흐름과 정보 구조를 설계했습니다.

01. 브랜드 첫인상 전달

사용자가 사이트에 진입하는 순간
브랜드만의 감성과 분위기를 경험할 수 있도록
비주얼 중심의 히어로 섹션을 구성했습니다.

03. 라이프스타일 속 제품 발견

실제 사용자의 라이프스타일 이미지를 활용하고
이미지에 사용된 제품 정보를 함께 제공하여
브랜드 경험과 제품 탐색이 자연스럽게
이어질 수 있도록 설계했습니다.

02. 제품 탐색 경험

호버 시 제품을 사용하는 모델 컷과
빠른 장바구니 추가 버튼을 제공하여
사용자가 제품 사용 모습을 직관적으로
확인하고 구매 행동으로 자연스럽게
이어질 수 있도록 설계했습니다.

02

Brand Story

브랜드의 철학과 감성을 직접 경험할 수 있도록
스토리텔링 중심의 페이지로 구성했습니다.

01. 브랜드 철학 전달

ODE가 추구하는 가치와 브랜드 스토리를
감성적인 이미지와 텍스트로 풀어내어
사용자에게 자연스럽게 전달합니다.

02. 제품 연결

브랜드 스토리에서 자연스럽게
제품 탐색으로 이어지는 흐름을
구성하여 전환율을 높였습니다.
